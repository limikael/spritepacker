Spriter
=======

Command line tool and grunt task to generate sprite sheets.

Command line tool
-----------------

Install with:

```
npm install -g spriter
```

Run with:

```
spriter my_file.png my_other_file.png
```

This will generate `spritesheet.json` and `spritesheet.png` in the current directory.
